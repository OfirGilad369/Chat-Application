# Chat Application

### Usage:

##### 1. in path "/chatServer" run:
```node server/chat-server.js```

##### 2. in path: "/chat" run:
```npm start```

### Installed libraries:
##### 1. formik 
##### 2. redux 
##### 3. react-redux 
##### 4. react-router-dom
##### 5. redux-thunk 
##### 6. redux-persist
##### 7. bootstrap
##### 8. jquery
##### 9. popper.js
